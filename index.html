RMS Components Styleguide

This styleguide is intended to provide examples of key components, for use in RMS forms. The components are originally provided by Bootstrap, which has then been customised slightly with UI tweaks and custom colours.

The rms.css file inside dist/css contains the compiled Bootstrap CSS + RMS customisations, and needs to be included in projects using these components.

Every component here is compatible with Bootstrap's customisation capabilities, such as sizing and colours. This styleguide does not go through all of the possible examples, as Bootstrap details them way better. To see them all, < target="_blank"a href="http://getbootstrap.com/docs/4.1">view Bootstrap's full documentation, or click the Bootstrap link below each component's title.

Note: Some of the examples are inside containers with a fixed width, to prevent them from spanning unusually wide.

Steps Progress Bar

This has been created on top of the existing < target="_blank"a href="http://getbootstrap.com/docs/4.1/components/pagination/">Bootstrap Pagination component, meaning the optional styles you can add are the same.

Each step is a link, meaning they could be clicked to go back or forward to that particular step.

For the content inside the each step, we would recommend using icons. We've included Font Awesome in this styleguide as an example of icons that could be used.
